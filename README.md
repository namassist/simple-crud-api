# SIMPLE CRUD API USING REACT JS

### Requirements
1. Node JS
2. MYSQL
****


### SET UP BACKEND
1. Import Database politeknik.sql
2. Open project with text editor
3. Install dependency using command `npm install`
4. Run server using command `node index.js`
****

### SET UP Frontend
1. Open project with text editor
2. Install dependency using command `npm install`
3. Run server using command `npm run dev`
4. Open Browser with URL `http:localhost:3000`
